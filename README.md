# 🚀 OnePager Website - React + Vite + Tailwind CSS

A modern, responsive one-page website built using **React.js**, **Vite**, and **Tailwind CSS**.  
This project was developed as part of my frontend development practice to improve component-based architecture, responsive design, and UI implementation skills.

---

## 🌐 Live Demo

🔗 **Live Website:** https://task2-ml-bench-one-pager-design-w6j.vercel.app/

🔗 **Source Code:** https://github.com/beenishishaq2026-iss/Task2-MLBench-OnePagerDesign

## 📌 Project Overview

This project is a responsive single-page website that recreates a professional landing page design using React components and Tailwind CSS.

The goal of this project was to practice:

- React component structure
- Responsive layouts
- Tailwind CSS styling
- Reusable UI sections
- Modern frontend development workflow

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/beenishishaq2026-iss/Task2-MLBench-OnePagerDesign.git
```

Navigate to the project folder:

```bash
cd Task2-MLBench-OnePagerDesign
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
